## Обновление

Новую версию можно устанавливать поверх старой. Удалять приложение перед обновлением не нужно.

При обычном обновлении сохраняются:

- вход в аккаунт;
- выбранный сервер;
- настройки приложения;
- правила трафика.

Если Android сообщает, что приложение имеет другую подпись, значит старая версия была установлена из другого источника или была тестовой сборкой. Только в этом случае потребуется удалить старую версию и установить актуальную заново.
